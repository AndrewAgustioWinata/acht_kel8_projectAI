Cara menjalankan program Chatbot Translation :
===============================================
1. Lakukan "npm install" terlebih dahulu untuk menginstall seluruh dependencies yang tersedia, yang terdiri dari :
- Telegraf : library untuk membuat bot Telegram menggunakan Node.js.
- Nodemon : tool yang membantu pengembangan aplikasi Node.js dengan secara otomatis me-restart aplikasi saat perubahan file terdeteksi.
- Tesseract : library OCR (Optical Character Recognition) yang memungkinkan konversi teks dari gambar menjadi teks yang dapat dibaca oleh mesin. 
- Translation Google : library wrapper untuk API Google Translate, yang memungkinkan penerjemahan teks antarbahasa.
- GTTS (Google Text to Speech) : library yang digunakan mengonversi teks menjadi suara menggunakan layanan Google Text-to-Speech API.

2. Buka terminal pada directory, lalu jalankan "npm run bot" untuk menjalankan program.

3. Klik link berikut untuk masuk ke room chat
http://t.me/kelompok8_translation_bot

4. Klik tombol start untuk memulai proses penerjemahan dengan bot. (NB: Jika tombol start tidak tersedia, ketik "/start" sebagai alternatif.
