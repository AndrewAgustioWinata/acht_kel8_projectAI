Cara menjalankan program Gamebot Pacman :
=========================================
1. Download Unity Editor.
2. Download Unity 2022.3.14f1 (version sensitive).
3. Buka file location menggunakan Unity Editor.
4. Di Tab Game terdapat tombol "Play" (di bagian tengah atas) atau gunakan "Ctrl + P" di Tab Game untuk memulai permainan.
5. Gunakan tombol "W" "A" "S" "D" untuk mengendalikan pergerakan saat bermain.
